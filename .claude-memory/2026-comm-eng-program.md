## 核心决策

1. **毕业要求框架**：使用2024版11条（非旧版12条），旧6+旧7合并为"工程与可持续发展"，旧8升级为"工程伦理和职业规范"
2. **"计算"定位**：4.3.1工程知识新增"计算"作为独立知识来源（与数学、自然科学并列），不等同于4.3.5使用工具
3. **三个职业出口**：
   - 出口一：AI应用与智能系统（大模型应用、智能体开发，替代传统互联网软件开发）
   - 出口二：通信硬件与算法（FPGA、嵌入式、软硬件协同）
   - 出口三：智能体通信与无人系统通信（多智能体协作、无人机集群组网、卫星互联网、量子通信）
4. **二级指标点**：37个，每条毕业要求2-4个
5. **与2022版对比**：删除"项目管理与经济决策"、"工程伦理和社会责任"独立课，相关支撑分散到其他课程（这是潜在风险点）
