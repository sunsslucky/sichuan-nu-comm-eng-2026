## 核心决策

1. **毕业要求框架**：使用2024版11条（非旧版12条）
   - 旧6（工程与社会）+旧7（环境与可持续发展）合并为 `4.3.6 工程与可持续发展`
   - 旧8（职业规范）升级为 `4.3.7 工程伦理和职业规范`（强调工程报国、践行工程伦理）
2. **"计算"定位**：4.3.1工程知识新增"计算"作为独立知识来源，与数学、自然科学并列。计算 ≠ 使用工具。计算 = 计算思维（算法设计、复杂度分析、计算建模），核心是"怎么想问题"；使用工具 = 会操作软件/仪器。
3. **三个职业出口**：
   - 出口一：AI应用与智能系统（大模型应用、智能体开发、AI系统集成，**替代传统互联网软件开发**）
   - 出口二：通信硬件与算法（FPGA、嵌入式、软硬件协同）
   - 出口三：智能体通信与无人系统通信（多智能体协作通信、无人机集群组网、低空通信、卫星互联网、量子通信）
4. **二级指标点**：37个，每条毕业要求2-4个
5. **与2022版对比的关键风险**：
   - 删除了"项目管理与经济决策"独立课 → 4.3.10项目管理支撑偏弱
   - 删除了"工程伦理和社会责任"独立课 → 4.3.7工程伦理支撑偏弱
   - 删除了"英文文献阅读" → 4.3.2问题分析中文献调研能力支撑减弱
   - 补充标准提到"含离散数学"，2026版未设置

## 电子信息与电气工程类补充标准要点

通信工程必须覆盖的知识链条：物理机制 → 电子线路 → 信号/信息的获取、分析、存储和传输 → 系统和网络。培养硬件+软件+系统的分析与设计能力。数学应含离散数学。
